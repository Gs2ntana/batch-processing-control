Este projeto é um sistema de gerenciamento de lotes que permite cadastrar, listar, atualizar e remover lotes de forma eficiente.

O back-end foi desenvolvido em Java, responsável pelas regras de negócio e pela exposição de uma API REST, enquanto o front-end utiliza JavaScript com Fetch API para consumir os endpoints e realizar a comunicação assíncrona entre cliente e servidor.

O objetivo do projeto é aplicar conceitos de integração entre front-end e back-end, boas práticas de APIs REST e manipulação de dados em um cenário real de gerenciamento.
